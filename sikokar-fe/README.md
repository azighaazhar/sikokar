# Sikokar Frontend

Frontend Next.js untuk Sikokar.

## Prasyarat
- Node.js 18+ (disarankan 20+)

## Menjalankan Lokal
1. Salin `.env.example` menjadi `.env.local`.
2. Install dependencies:
	```bash
	npm install
	```
3. Jalankan dev server:
	```bash
	npm run dev
	```
4. Buka `http://localhost:3000`.

## Env
- `NEXT_PUBLIC_API_BASE_URL` (contoh `http://localhost:3002`)
